📘 Experiment No. 08 – CDMA

📌 Aim

To implement CDMA and test autocorrelation and orthogonality of codes.

🧠 Theory

CDMA (Code Division Multiple Access) is a technique that allows multiple users to share the same communication channel using unique codes. It uses spread spectrum technology to reduce interference.

Autocorrelation helps in signal detection and synchronization

Orthogonality ensures signals do not interfere with each other

🛠️ Tools

Android Studio / Java

CDMA Algorithm

📊 Output

CDMA implemented successfully

Autocorrelation and orthogonality verified

📌 Conclusion

The experiment successfully demonstrated CDMA working and its properties.
